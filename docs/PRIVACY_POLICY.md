# 回声破除者 隐私协议与数据合规文档

**版本**：v1.0
**生效日期**：2026年5月22日
**适用范围**：回声破除者浏览器插件（Edge / Chrome）

---

## 一、数据收集原则

### 1.1 本地优先原则
回声破除者遵循**"本地优先"**数据策略：
- 所有行为统计数据默认仅存储在用户本地浏览器的 `chrome.storage.local` 中
- 不主动上传任何用户行为数据至云端服务器
- 仅在用户明确授权后，才会将特定数据加密传输至云端

### 1.2 最小必要原则
仅收集实现产品核心功能所必需的最少数据：
- AI 网站使用时长（秒级精度）
- 连续提问轮数
- 复制粘贴操作次数
- 触发唤醒事件的时间戳
- 用户对唤醒提示的响应（忽略/暂停思考）

### 1.3 不收集的数据
本插件**明确不收集**以下数据：
- 用户在 AI 网站上输入的具体问题内容
- AI 返回的具体回答内容
- 用户的浏览历史（非 AI 网站的访问记录）
- 用户的个人身份信息（姓名、身份证号等）
- 用户的账户凭证

## 二、数据存储说明

### 2.1 本地存储结构

| 存储键 | 数据内容 | 保留期限 |
|--------|----------|----------|
| `echo_breaker_usage` | 每日使用统计（时长、轮数、复制次数） | 本地持久保存，用户可手动清除 |
| `echo_breaker_settings` | 用户偏好设置 | 本地持久保存 |
| `echo_breaker_prompts` | 苏格拉底文案库 | 本地持久保存，支持远程更新 |
| `echo_breaker_triggers` | 触发事件日志 | 本地持久保存 |
| `echo_breaker_thoughts_YYYY-MM-DD` | 思考日志（L3） | 本地持久保存，用户可手动清除 |

### 2.2 云端存储（需用户授权）

以下数据仅在用户**独立勾选"同意上传训练数据"**后才会传输：

| 数据类型 | 加密方式 | 用途 |
|----------|----------|------|
| 思考日志 | AES-256 | 思维偏差分析 |
| 靶场答案 | AES-256 | 薄弱依据比对 |
| 匿名化使用统计 | HTTPS | 产品优化 |

### 2.3 数据脱敏规则
- 社群公开内容不包含可识别身份信息
- 用户名随机生成（如"思考的企鹅#3847"）
- 上传数据前自动移除所有 URL 中的查询参数和路径信息
- 时间戳仅保留日期级别，不包含精确到秒的时间

---

## 三、数据传输安全

### 3.1 传输加密
- 全站 HTTPS 加密传输
- 敏感字段使用 AES-256 加密后再传输
- API 通信使用 JWT Token 认证

### 3.2 传输内容限制
- 仅传输脱敏后的统计数据
- 不传输用户原始输入文本（除非用户主动在 L3 思考日志中输入并授权上传）
- 不传输 AI 网站的原始 HTML 内容

---

## 四、用户权利

### 4.1 知情权
- 插件安装时展示完整隐私协议
- 首次使用时弹出数据收集说明
- 每次数据上传前确认提示

### 4.2 选择权
- 用户可随时关闭插件（全局开关）
- 用户可单独关闭数据上传功能
- 用户可选择干预强度（严格/建议/温和/轻干预）

### 4.3 访问权
- 用户可随时在 Popup 看板中查看自己的使用数据
- 用户可在设置页面导出全部本地数据

### 4.4 删除权
- 用户可手动清除全部本地存储数据
- 用户可申请删除云端数据（通过设置页面提交请求）
- 卸载插件时自动清除全部本地数据

### 4.5 撤回权
- 用户可随时撤回数据上传授权
- 撤回后立即停止数据上传，已上传数据将在30天内删除

## 五、第三方服务

### 5.1 大模型 API 调用
本插件在以下功能中调用大模型 API：
- L2 延迟满足层：生成引导式反问
- L3 元认知外显层：思维偏差分析
- L4 逆向重构层：薄弱依据判断

**调用规则**：
- 仅传输用户主动输入的思考内容和靶场答案
- 不传输用户在 AI 网站上的原始对话
- 每用户每日 API 调用上限 50 次
- API 供应商：DeepSeek / 智谱 GLM / 通义千问（按可用性自动切换）

### 5.2 第三方数据存储
- 云端数据存储于国内云服务商（阿里云/腾讯云）
- 遵守《中华人民共和国数据安全法》和《个人信息保护法》

## 六、合规声明

### 6.1 法律合规
- 遵守《中华人民共和国网络安全法》
- 遵守《中华人民共和国数据安全法》
- 遵守《中华人民共和国个人信息保护法》
- 遵守《信息安全技术 个人信息安全规范》(GB/T 35273)

### 6.2 浏览器商店合规
- 遵守 Microsoft Edge Add-ons 开发者政策
- 遵守 Chrome Web Store 开发者计划政策
- Manifest V3 合规：无远程代码执行、无持久化 Background Page

### 6.3 教育场景合规
- 面向高校使用时遵守教育部《高等学校信息公开办法》
- B端产品不收集学生成绩等敏感教育数据
- 教师端仅提供聚合统计数据，不展示个体原始数据

---

## 七、未成年人保护

- 本产品不面向14岁以下未成年人
- 如发现14岁以下用户使用，将通知监护人并限制功能
- 教育场景下，学生数据仅用于认知训练，不用于评价或排名
